########################
 Contributing to WarMAC
########################

|  Contributing to WarMAC is simple. As WarMAC is a hobby project of mine that
   I work on in my spare time, WarMAC is rapidly changing. This means that as
   of right now, feature contributions from others are likely to be rejected.

   Don't fret though. WarMAC will likely see a stable release at some point
   within the next year!
